# mini-homm-web

一个类《英雄无敌》网页原型。

当前版本是更完整的可玩原型，包含：

- 20x20 格子地图
- 多英雄移动与切换
- 资源拾取
- 矿点占领
- 回合推进
- 每周兵力增长
- 城堡招募、议事厅、兵营和酒馆
- 敌军回合与动态刷新
- 英雄经验、升级和神殿强化
- 战役胜负目标
- 带损耗预估的简化战斗结算

## 在线地址

启用 GitHub Pages 后，访问：

```text
https://maning711.github.io/mini-homm-web/
```

## 本地预览

直接在仓库根目录起静态服务器即可：

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://127.0.0.1:8000/
```
