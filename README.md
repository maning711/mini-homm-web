# mini-homm-web

一个类《英雄无敌》网页原型。

当前版本是最小可玩稿，包含：

- 20x20 格子地图
- 英雄移动
- 资源拾取
- 矿点占领
- 回合推进
- 每周兵力增长
- 城堡招募
- 极简战斗结算

## 在线地址

启用 GitHub Pages 后，访问：

```text
https://maning711.github.io/mini-homm-web/
```

## 本地预览

直接在仓库根目录起静态服务器即可：

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://127.0.0.1:8000/
```
